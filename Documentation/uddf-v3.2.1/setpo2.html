setpo2

Occurrence

optional, multiple

Attributes

setby=user | computer — compulsory

Description

Via <setpo2> the maximum tolerated oxygen partial pressure chosen by the diver (set attribute setby="user"), or the dive computer (set attribute setby="computer") is set for diving with rebreathers. The value is given as a real number in the SI unit Pascal and remains constant as long as a new value is set with another <setpo2> statement. Hereby, set values by the user, and the dive computer are to be treated independent.

Parent Elements

<waypoint>

Child Elements

—

See Also

<calculatedpo2>, <measuredpo2>, <samples>, <waypoint>

Examples

(see also <samples>) 

<!-- user set max PO2 of 1.4 bar -->
<setpo2>1.4e5</setpo2>
