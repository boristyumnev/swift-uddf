calculatedpo2

Occurrence

optional, single

Attributes

—

Description

The element indicates the oxygen partial pressure calculated by a dive simulation software, given in the SI unit Pascal as a real number.

Parent Elements

<waypoint>

Child Elements

—

See Also

<measuredpo2>, <setpo2>, <waypoint>

Examples

<!-- a PO2 of 1.27 bar was calculated by the dive simulation program -->
<calculatedpo2>1.27e5</calculatedpo2>
