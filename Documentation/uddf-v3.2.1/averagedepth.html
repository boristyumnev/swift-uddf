averagedepth

Occurrence

optional, single

Attributes

—

Description

The element indicates the average depth in metres (given as a real number) of the dive, calculated by the <divecomputer> using the recorded profile data.

Parent Elements

<informationafterdive>

Child Elements

—

See Also

<averagevisibility>

Examples

(see also <informationafterdive>) 

<!-- average depth of the dive is 11.8 m -->
<averagedepth>11.8</averagedepth>
