depth

Occurrence

optional/compulsory (depending on context), single

Attributes

—

Description

Depth statement given as a real number in metres for a <waypoint>.

Normally, every <waypoint> must contain one <depth> statement. The only exception from this rule is the calculation of an ascent profile (see <calculateprofile>>) — because in this case the profile is to be calculated, i.e. time/depth values.

Parent Elements

<waypoint>

Child Elements

—

See Also

<waypoint>

Examples

(see also <waypoint>) 

<!-- a dive depth of 18,4 metres -->
<depth>18.4</depth>
