otu

Occurrence

optional, single

Attributes

—

Description

<otu> sets the "oxygen toxicity units" during the dive, given as a real number.

The OTU value is considered to be constant until a next <otu> statement alters it.

Parent Elements

<waypoint>

Child Elements

—

See Also

<samples>, <waypoint>

Examples

(see also <samples>) 

<otu>500.0</otu>
