nodecotime

Occurrence

optional, single

Attributes

—

Description

Indicates the amount of the No Decompression Time at the given time and depth calculated by the dive computer, or a dive simulation program respectively; given as a real number in seconds.

Parent Elements

<waypoint>

Child Elements

—

See Also

<decostop>, <waypoint>

Examples

(see also <waypoint>) 

<!-- 90 minutes no decompression time -->
<nodecotime>5400.0</nodecotime>
