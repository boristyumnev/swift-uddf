desaturationtime

Occurrence

optional, single

Attributes

—

Description

The desaturation time based on the given diving profile calculated by the dive computer, or a dive simulation program respectively, is put into brackets via <desaturationtime>. The value is given as a real number in seconds.

Parent Elements

<informationafterdive>

Child Elements

—

See Also

<dive>, <informationbeforedive>, <noflighttime>

Examples

(see also <dive>) 

<!-- desaturation time 9 h 18 min -->
<desaturationtime>33480.0</desaturationtime>
