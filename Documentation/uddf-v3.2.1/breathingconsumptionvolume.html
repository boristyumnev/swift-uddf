breathingconsumptionvolume

Occurrence

optional, single

Attributes

—

Description

Indicates the breathing gas consumption volume for this breathing gas. The unit to be used is cubic metre per second [m^3/s]! A conversion into the unit "litre per minute" which is normally used in dive tables is subject to the executing software — UDDF uses SI units only!

Parent Elements

<tankdata>

Child Elements

—

See Also

<tankdata>

Examples

<!-- BCV of 20 l/min ^= 0.00033333... m^3/s -->
<breathingconsumptionvolume>0.000333333</breathingconsumptionvolume>
