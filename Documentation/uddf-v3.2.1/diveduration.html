diveduration

Occurrence

compulsory, single

Attributes

—

Description

Via <diveduration> the duration of the dive is given in seconds as a real number. <diveduration> must be given even if the profile data are stored inside <samples> (from which the duration can be calculated).

Parent Elements

<informationafterdive>

Child Elements

—

See Also

<dive>, <informationafterdive>

Examples

see also <dive> 

<!-- duration of dive 1 h 13 min -->
<diveduration>4380.0</diveduration>
